Use the editor below to author/update Multiple Choice, Single Select question. Typical sequence of authoring and options include the following:
  • Add Instructions [mandatory]
    This should include the question text to be presented to the student and additional context as necessary.
  • Define Choices [mandatory]
    Choices to be presented to the student a options (possible answers). Use add/edit/remove/reorder options to appropriate setup choices.
  • Mark Correct Answer [mandatory]
    Simply mark the correct answer by selecting the corresponding radio button (for a choice).
  • Specify Feedback [optional]
    Optional instructor feedback may be shown to the user after the answer is submitted (correct or incorrect) or when user request help/hints.
